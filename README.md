TRUE STYLE is a full-stack e-commerce web application built using the MERN stack (MongoDB, Express.js, React.js, and Node.js).  
The project includes both a User Panel and an Admin Panel with secure authentication and authorization.

## Live here : https://e-commerce-frontend-five-swart.vercel.app/

## Project Overview

TRUE STYLE is an online clothing store platform where users can browse products, add them to cart, and place orders using Cash on Delivery (COD).  
Admins can manage products and orders through a protected admin dashboard.

---

## Technologies Used

### Frontend
- React.js (Vite)
- Tailwind CSS
- React Router DOM
- Axios
- React Toastify

### Backend
- Node.js
- Express.js
- MongoDB with Mongoose
- JSON Web Token (JWT)
- Bcrypt.js
- Multer (Image Upload)

---

## User Features

- User registration and login
- JWT-based authentication
- Browse products by category
- Add products to cart
- Update cart quantity
- Place orders using Cash on Delivery
- View order history
- Secure protected routes

---

## Admin Features

- Admin login
- Add, update, and delete products
- Upload multiple product images
- View all orders
- Update order status
- Admin-only protected APIs

---

## Authentication & Authorization

- JWT token-based authentication
- Role-based access control (User / Admin)
- Secure middleware for protected routes
- Token stored in browser localStorage
